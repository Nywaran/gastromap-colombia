Project Overview

🎯

Objective

Design and implement an intelligent system that identifies Colombian municipalities with high potential for gastronomic tourism by leveraging publicly available climatic and geographic datasets, combined with the National Tourism Registry (RNT).

📊

Data Sources

Three open data sources are merged: DIVIPOLA (DANE) for geographic coordinates of 1,122 municipalities, the RNT (MinCIT) with 686,967 tourism service providers, and a climate estimation model based on Colombia's thermal floor system.

🤖

Machine Learning Approach

We apply feature engineering with log-transformations and min-max normalization, a weighted composite scoring model to build the GTPI, and binning classification to group municipalities into five tiers from "Very Low" to "Very High" potential.

🗺️

Interactive Visualization

The results are displayed on a dark-themed Leaflet.js map with color-coded markers, filterable by region, department, and GTPI range. A REST API serves the processed data for programmatic access.

⚙️

Technology Stack

Built with Python 3, Flask (web framework), Pandas & NumPy (data processing), Leaflet.js (mapping), and the datos.gov.co API for real-time data access.

📈

Key Findings

Bogotá (95.1), Cartagena (89.4), and Medellín (83.1) lead the GTPI ranking. The Andean and Caribbean regions concentrate 85% of all gastronomy establishments. Only 371 out of 1,015 active municipalities have registered gastronomy, revealing significant development opportunity.
