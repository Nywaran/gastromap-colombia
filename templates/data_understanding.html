Data Quality Evaluation

Completeness ✓

DIVIPOLA has zero null values in key columns. RNT has minimal missing data in non-critical fields. All join keys (municipality codes) are fully populated.

Consistency ✓

Municipality codes follow the DIVIPOLA standard (5 digits). After zero-padding normalization, cross-dataset joins work reliably with 100% key match rate.

Volume ✓

686,967 RNT records and 1,122 municipalities far exceed the minimum 1,000-record threshold. The dataset provides statistical significance for all analyses.

Timeliness ⚠

RNT data spans 2019-2026 and was last updated May 2026. DIVIPOLA coordinates are current. Climate data is estimated, not real-time.

Correlation Analysis

A correlation matrix was computed for all numeric variables in the RNT dataset. The key finding is that NUMERO_DE_EMPLEADOS (employee count) correlates moderately with establishment size and category, confirming it as a valid proxy for sector maturity. Geographic codes (department, municipality) show no meaningful correlation with operational variables, as expected — location codes are identifiers, not predictive features.

The full correlation matrix is available via the /api/stats endpoint and was used to validate that the five GTPI components capture independent dimensions of gastronomic tourism potential.
