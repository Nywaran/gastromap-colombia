Description

{% if m.name == 'Random Forest' %}

An ensemble method that builds multiple decision trees on random subsets of data and features, then combines their predictions by majority voting. It handles non-linear relationships well and provides feature importance rankings. class_weight='balanced' compensates for the imbalanced dataset by adjusting sample weights inversely proportional to class frequency.

{% elif m.name == 'Gradient Boosting' %}

A sequential ensemble method that builds decision trees one at a time, each correcting the errors of the previous one. It optimizes a loss function via gradient descent in function space. Generally achieves higher accuracy than Random Forest but is more prone to overfitting. The learning_rate=0.1 controls how much each tree contributes, balancing speed vs accuracy.

{% elif m.name == 'K-Nearest Neighbors' %}

A lazy learning algorithm that classifies new samples based on the majority class of their K nearest neighbors in feature space. weights='distance' gives closer neighbors more influence. It makes no assumptions about data distribution but is sensitive to feature scaling (hence StandardScaler). Simple and interpretable but slower for large datasets.

{% endif %}

All three models exceed 96% accuracy, validating that the engineered features contain strong signal for predicting gastronomic tourism potential class.

The fact that supervised models can learn the GTPI classification with high accuracy confirms that our unsupervised scoring formula captures genuine patterns in the data, not noise.
